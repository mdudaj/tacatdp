# TACATDP Power Pages Package

This package contains the downloaded Power Pages site source and will host the ODK Web Forms SPA package.

- `tacatdp-monitoring-tool/.powerpages-site/`: PAC-downloaded Power Pages site source.
- `webforms-spa/`: Vue/Vite SPA that integrates ODK Web Forms and calls Power Pages `/_api`.

Do not edit or upload site content without confirming the active environment and website ID.
