# User Stories: Canvas Metadata Renderer MVP

## CANVAS-US-01 Assigned forms

As a data collector, I want to see only active forms assigned to me so that I know what I need to complete.

## CANVAS-US-02 Start or resume submission

As a data collector, I want to start a new submission or resume an existing draft so that I do not lose work.

## CANVAS-US-03 Render seeded questions

As a data collector, I want the app to render the seeded form questions from Dataverse so that the same app can run future forms.

## CANVAS-US-04 Save draft

As a data collector, I want to save a draft so that I can leave and return before submitting.

## CANVAS-US-05 Submit

As a data collector, I want required validation before submit so that incomplete submissions are not accepted.

## CANVAS-US-06 Edit until locked

As a data collector, I want to edit submitted records until they are locked so that I can correct mistakes.

## CANVAS-US-07 Attach evidence

As a data collector, I want to attach one photo/file to the form so that evidence is linked to the submission.

## CANVAS-US-08 Capture GPS

As a data collector, I want to capture a GPS point when available so that location evidence can be stored with the submission.

## CANVAS-US-09 View history

As a data collector, I want to see my submission history for the selected form so that I can review status and reopen editable records.

## CANVAS-US-10 Handle errors

As a data collector, I want clear messages when save or submit fails so that I know whether to retry or ask for support.
