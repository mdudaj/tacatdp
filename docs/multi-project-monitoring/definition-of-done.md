# Definition of Done: Multi-Project Monitoring Model

This research/planning slice is done when:

1. Research, requirements, PRD, data model, controlled vocabulary, stories, acceptance criteria, traceability, readiness, delivery plan, and verification artifacts exist.
2. Artifacts cite ODK, OpenClinica/CDISC, REDCap, and InvenioRDM-derived patterns.
3. TACATDP is defined as the first project configuration, not the hard-coded platform schema.
4. Next action is clear: generate reviewable Dataverse schema artifacts.
5. No Dataverse, SharePoint, app publish, or permission-changing command is run.
6. Handoff is refreshed.
