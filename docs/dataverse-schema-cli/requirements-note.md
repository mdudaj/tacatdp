# Requirements Note: Dataverse Schema CLI Delivery

The next TACATDP MVP task is to prepare and execute the dev Dataverse schema setup for the ten MVP tables.

The implementation must remain dry-run-first. It may generate a Web API operation plan and reviewed deployment commands, but must not write to Dataverse unless the target environment, solution name, schema artifacts, and exact command are explicitly approved.

The delivery should prove schema readiness for the Canvas metadata renderer, not the full ODK/XLSForm platform.
