# Definition of Done: Dataverse Schema CLI Delivery

This delivery slice is done when:

- Preflight validates `.env.example` and local `.env` without exposing secrets.
- The schema plan lists only the ten MVP tables and reviewed columns.
- The approved dev write path is documented and gated.
- The implementation refuses production and destructive operations by default.
- One small seed form plan is ready for the Canvas renderer.
- Verification commands are documented.
